About :

What is Project Redesign?

Its an initiative designed to bring certain popular but old looking websites to a new modern looking interface with the help of browser extensions. These extensions are designed to leverage the latest browsers so I recommend using them on the latest updated browsers.

The first extensions made under Project Redesign is Wiki Redesigned. Which bring a modern look to Wikipedia and its related sites.

What are the Requirements?

The only requirements are that you use a chromium or firefox based browser, the extensions was designed keeping the latest version of chrome or firefox.

The extension should also work on any chromium based browser like 'Edge-chromium', 'Opera' etc. The extensions are Desktop Only for now.

For example: The extention 'Wiki Redesigned' only works on desktop website. That means it will work on "www.en.wikipedia.org"but will not work on "www.en.m.wikipedia.org".

Performance Impact?

The extension won't cause any noticeable impact on performance in any modern PC.

Is it Open Source?

YES. The code will be hosted on Github (link).
