User & Project Menus

The project menu has 5 options :

DarkTheme :

To toggle dark mode.

MonoText :

All Text will have same color.

References :

To toggle reference classes.

PureRead :

To toggle reading mode. It also automatically toggle the "MonoText" & "References" options.

ImageViewer :

To toggle new image viewer.
