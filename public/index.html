About :

What is Project Redesign ?

Project Redesign is an initiative to redesign certain websites for the online community. The Redesignes are provided through browser extentions. These extentions are mainly for modern browsers and may not work properly on old browsers, Thus I recommend using latest version of browsers. The first project under this initiative is "Wiki-Redesign", which provide redesigned wikipedia.

What are the Requirments ?

There are no special requirments , you just need the latest verison of chrome or firefox. The extention should also work on any chromium based browser like 'Edge-chromium', 'Opera' etc. The extentions are Desktop Only for now.

For example: The extention 'Wiki Redesigned' only works on desktop website. That means it will work on "www.en.wikipedia.org"but will not work on "www.en.m.wikipedia.org".

Will it take more time to load the web pages of redesigned website ?

Yes, it will, but the aditional time should just be in the range of few miliseconds. Most of the time it will not be noticible to you, unless you are loading a page with huge amount of information/contents.

Is it safe ?

Yes! it's completely safe. Its just a basic software there is nothing fishy going on
